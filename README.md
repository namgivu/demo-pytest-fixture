# demo-pytest-fixture


This demo will run:

    - TC1: No yield in pytest.fixture
        + Start NO YIELD
        + STOP NO YIELD
        + Run TC 1

    - TC2: local pytest.fixture
        + Start LOCAL fx
        + Run TC 2
        + Done LOCAL fx

    - TC3: share pytest.fixture
        + Start SHARE fx
        + Run TC 3
        + Done SHARE fx
